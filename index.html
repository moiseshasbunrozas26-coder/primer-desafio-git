Descripción del proyecto

Este sitio web forma parte de un ejercicio práctico sobre Git y GitHub. El repositorio registra cada etapa de desarrollo mediante commits y ramas.

Objetivos del proyecto

  • Crear un repositorio local.
  • Utilizar HTML semántico.
  • Registrar cambios con commits claros.
  • Trabajar con ramas de desarrollo.
  • Integrar cambios en la rama principal.
