# Primer desafío: Git y GitHub

## Descripción

Este repositorio corresponde al primer taller formativo de la asignatura Aplicaciones y Tecnologías de la Web.

El proyecto tiene como propósito aplicar los fundamentos de Git y GitHub mediante la creación de un repositorio, el registro de cambios con commits, el trabajo con ramas y la integración de nuevas funcionalidades.

Estado del proyecto: versión estable con mejoras de desarrollo

## Objetivos

- Inicializar y configurar un repositorio Git.
- Crear una página web con HTML semántico.
- Registrar cambios mediante commits descriptivos.
- Trabajar con una rama de desarrollo.
- Fusionar cambios en la rama principal.
- Resolver un conflicto básico de Git.

## Estructura del repositorio

- `README.md`: documentación general del proyecto.
- `index.html`: página principal del sitio web.
- `funcionalidad.html`: página creada desde la rama desarrollo.

## Tecnologías utilizadas

- HTML5
- Git
- GitHub
- PowerShell
